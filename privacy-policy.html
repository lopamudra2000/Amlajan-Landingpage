Private Data We Receive And Collect

The Amlajan app is a app where it provides all the details about the nearby oxygen cylinders providers in your locality. It requires basic information like name, address of the patient and the name, address, phone number of the provider. This data will only shared with the users using the app. For address, the app will require location services and for contacting the provider, the app will require call and message services. The data is not used by any third-party services for advertisements. It is a free to use app that has contains no in-app advertisements.
